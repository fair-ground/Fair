Instance Method

elementDictionary(attributes:childNodes:)

Converts the current node into a dictionary of element children names and the trimmed content of their joined string children. Note that any non-content children are ignored, so this is not a complete view of the element node.

Declaration

func elementDictionary(attributes: Bool, childNodes: Bool) -> [String : String]

Discussion

E.g. the XML:

<ob><str>X</string><num>1.2</num></ob>

will return the dictionary:

["str": "X", "num": "1.2"]
